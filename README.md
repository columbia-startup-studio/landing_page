# Circle Landing Page

This is the responsive, static landing page for Circle—built with React and Vite. It serves to collect waitlist signups and explains the core features of the "Stop Texting. Start Meeting." app.

## Technologies Used
- **React 18**
- **Vite** (for fast, optimized builds)
- **Lucide React** (for feather icons)
- **Vanilla CSS** (for precise, lightweight styling matching the design tokens)

## Getting Started

Follow these steps to run the application locally.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed (version 18+ recommended).

### 1. Install Dependencies
Run the following command in the project root to install the required packages:
```bash
npm install
```

### 2. Start the Development Server
Once dependencies are installed, start the Vite development server:
```bash
npm run dev
```
The app will typically be available at `http://localhost:5173/`. 

### 3. Build for Production
To create an optimized production build of the static site:
```bash
npm run build
```
The compiled files will be output to the `dist/` directory, which can be deployed to any static hosting service (like Vercel, Netlify, or GitHub Pages).
